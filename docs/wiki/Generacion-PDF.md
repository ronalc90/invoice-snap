# Generacion de PDF

## Tecnologia

InvoiceSnap usa `@react-pdf/renderer` para generar facturas en PDF. Esta libreria permite definir el layout del PDF usando componentes React.

## Plantilla

La plantilla esta definida en `src/lib/pdf-template.tsx` y consta de:

### Estructura del PDF

1. **Header** -- Titulo "INVOICE", numero de factura, nombre del negocio
2. **Fechas** -- Fecha de emision, fecha de vencimiento, estado
3. **Bill To** -- Datos del cliente (nombre, empresa, email, direccion)
4. **Tabla de items** -- Descripcion, cantidad, precio unitario, monto
5. **Totales** -- Subtotal, impuesto (si aplica), total
6. **Notas** -- Notas adicionales (si las hay)
7. **Footer** -- "Generated by InvoiceSnap" + fecha

### Componente Principal

```typescript
interface InvoicePDFProps {
  invoice: InvoiceWithRelations;
  businessName?: string;
  businessAddress?: string;
}

function InvoicePDF({ invoice, businessName, businessAddress }: InvoicePDFProps)
```

El componente recibe la factura completa con sus relaciones (cliente, items, pagos) y los datos opcionales del negocio del usuario.

## Estilos

Los estilos se definen con `StyleSheet.create()` de React-PDF (no Tailwind CSS, ya que React-PDF tiene su propio sistema de estilos similar a React Native).

**Caracteristicas del diseno:**
- Tamano A4
- Fuente Helvetica (incluida en React-PDF)
- Color primario: `#2563eb` (azul)
- Texto secundario: `#6b7280` (gris)
- Secciones con fondo: `#f9fafb`
- Bordes sutiles en la tabla de items

## Formato de Moneda

El PDF usa `Intl.NumberFormat` para formatear montos:

```typescript
function formatCurrency(amount: number, currency: string = 'USD'): string {
  return new Intl.NumberFormat('en-US', {
    style: 'currency',
    currency,
  }).format(amount);
}
```

Soporta multiples monedas (USD, EUR, etc.).

## Configuracion en Next.js

React-PDF necesita configuracion especial para funcionar con Server Components:

```javascript
// next.config.js
const nextConfig = {
  experimental: {
    serverComponentsExternalPackages: ['@react-pdf/renderer'],
  },
};
```

Esto le indica a Next.js que no intente bundlear `@react-pdf/renderer` con el resto del codigo del cliente, ya que contiene modulos nativos de Node.js.

## Vista Previa

La vista previa de factura en el navegador (`/invoices/[id]/preview`) usa el componente `InvoicePreview` (HTML/Tailwind) en lugar del PDF. El PDF se genera bajo demanda para descarga.

El componente `InvoicePreview` replica visualmente la misma estructura del PDF pero usando HTML y Tailwind CSS para renderizacion inmediata en el navegador.
